pyweek-dojo
===========

A minimalistic versus fighting game (and a Pyweek 19 entry) 

![Screenshot](/resource/image/screenshot.png?raw=true "ScreenShot")

## Requirement

- Python 2.7
- Pygame 1.9

## Run locally

    $ python Dojo.py 
	# Or
	$ python -m dojo

## Installation

    $ python setup.py install

## Build executable

    $ pyinstaller Dojo.spec
	
## Controls

**Player 1:**
 - Move : WASD 
 - Jump : Tab
 
**Player 2:**
 - Move : Arrows 
 - Jump : RightShift
 
 **Controller:**
 - Move : Hat or Axis 
 - Jump : A (Button 0)
 - Reset: Y (Button 3)
 
**General:**
  - Reset : R
